Outline of the video course

Week 1 - BASH, HTML and Wireframes

  • Intro to Web Dev, Client vs Server model, career paths of a web dev
  • Before we jump into HTML, you will learn BASH, the linux programming language. After learning BASH, you can put on your resume that you know how to use linux machines
  • Introduction to HTML - In this short lesson you will learn all you need to know about HTML
  • You will learn how to make mockups using draw.io - it is crucial to wireframe your website before you write any code

Week 2 - CSS Basics

  • How to use Trello to manage your projects, many companies favor Trello over JIRA because of its UI and simplicity
  • Github crash course - absorb everything you need to know about using Github to save your code, including creating a repository, cloning a repo, creating branches and solving conflicts
  • Master the crucial CSS box model for margins and padding
  • Demystify the CSS display-block model and align elements easily
  • Master responsive design, a must-have skill in our mobile-dominated world
  • Manipulate font sizes using px and rem
  • Watch as students send me random websites and I re-create them from scratch using what we’ve learned

Week 3 - CSS Ninjitsu

  • CSS Advanced selectors - after you master this skill you will become a true CSS Ninja
  • Demystify making menu’s from CSS, now you can look at another website’s menu and say “Oh that’s how they did it”
  • CSS positions - how to position images and other elements
  • How to use the al-important flexbox in responsive design
  • ~120m - Learn Javascript basics
  • ~120m - Learn Javascript loops and logic

Week 4 - Javascript Ninjitsu

  • Learn more Javascript logic such as conditionals & while loops
  • How to use SEO properly on your website
  • Manage Javascript collections like arrays
  • How to use mouse events to interact with the user
  • How to use Bootstrap, the number one CSS & Javascript framework, for designing responsive websites - never design anything from scratch again
  • Learn PHP, your first server side language - generate dynamic, websites and take your first step to becoming a full stack ninja

Week 5 - PHP Power

  • Bootstrap Part 2 - We continue to drill Bootstrap skills until you know it by heart and this module also covers gradients to beautify your designs
  • PHP arrays - learn what arrays are and how to use them in different scenarios
  • PHP form processing - in previous modules you learned to make HTML forms, and now you will know what to do with the user submitted data
  • PHP Arrays & Form Processing
  • php $_GET & sessions
  • Interview advice - My students and I discuss what interviews are like and I give them interview advice

Week 6 - Advanced PHP

  • The difference between include and require keywords and how to use them to retrieve code from other files - this is essential if your code base starts to get large
  • Manipulating php multi-dimensional arrays
  • Introduction to jQuery, the number one JS language
  • Intro to jQueryUI, a framework for user interfaces in jQuery
  • Ajax - asynchronous javascript - how to get data without reloading a page
  • Introduction to MYSQL databases - how to use databases to store information permanently
